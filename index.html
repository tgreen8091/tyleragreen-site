Boston Realtime Map

Date: May 2016

Wanting to familiarize myself with the GTFS-realtime specification and Node.js applications, I created a realtime map of Boston transit!

You can read the full explanation on my blog. The code is available on GitHub!

GTFS to GeoJSON Ruby Gem

Date: March 2016

I tackled two birds with one stone on this project: 1) I learned the basics of creating and publishing my own Ruby gem, and 2) I implemented a simple algorithm to convert a GTFS feed into a GeoJSON file. I hope to be able to use this subproject in a larger transit visualization in the future!

You can read the full explanation on my blog. The code is available on GitHub and RubyGems.org!

New York City Transit Frequency Visualization

Date: February 2016

I created a web tool to visualize New York City transit frequencies using Transitland and Mapbox. Schedule data from buses and subways (and even a few ferry routes!) is plotted for an example Friday and Saturday morning. You can view any of the layers together to understand differences in service between modes and days.

You can read the full explanation on my blog. Feel free to explore the web tool itself!

Stadium Fanatic

Date: July 2010

I developed a basic social network around stadium travel in an effort to learn PHP and MySQL. As you will notice, I had not yet learned how to make a website look decent in the slightest.

Stadium Fanatic was my first foray into web programming and was critical to my understanding of the modern web. Even better, it sparked my passion for stadium travel. I would not be well on my way to visiting all 30 MLB ballparks without this formative project!
